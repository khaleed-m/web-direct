# Web Direct - AI Voice Agent Platform

Modern responsive website for AI voice agent platform with pricing plans and demo booking functionality.

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Sublime Text, etc.)
- Basic knowledge of HTML/CSS/JavaScript

### Clone Repository
```bash
git clone https://github.com/khaleed-m/web-direct.git
cd web-direct
```

### Run Locally
Simply open `index.html` in your web browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8000
```

Visit `http://localhost:8000` in your browser.

### 1. Company Branding
**File:** `index.html`, `pricing.html`, `booking.html`
- Change "Acme AI" to your company name
- Update page titles in `<title>` tags
- Modify navigation links if needed

### 2. Pricing Plans
**File:** `pricing.html`
- Update pricing tiers and features
- Modify plan names and costs
- Add/remove feature lists

### 3. Styling
**File:** `assets/style.css`
- Change colors, fonts, and layout
- Modify responsive breakpoints
- Update card designs and spacing

### 4. JavaScript Functionality
**File:** `js/convai-tools.js`
- Update form handling
- Modify API endpoints
- Add custom interactions

### 5. Configuration
**File:** `agent-schema.json`
- Update agent settings
- Modify conversation parameters
